# 北京租房市场数据分析

## 项目背景

本项目分析北京市租房数据，旨在探究租金分布、区域差异、房屋类型及价格模式。

目标是运用数据分析技术深入了解北京租房市场。

## 数据集

北京租房数据集

记录数量：7000+

字段包括：

- 价格

- 面积

- 户型

- 区/县

- 地铁距离

- 社区

## 工具

Python Pandas Matplotlib Seaborn Jupyter Notebook

## 主要分析

本项目探索以下内容：

1. 租金价格分布

2. 各区平均租金

3. 单元租金价格

4. 房屋类型分布

5. 面积与租金的关系

## 主要发现

- 海淀区单元租金价格最高

- 大部分出租房源为合租公寓

- 主要租金区间为2000-4000元人民币

- 地铁距离对租房需求有显著影响
